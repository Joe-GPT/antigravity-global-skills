# 內容策略指南 (Content Strategy)

## 1. Skill 名稱

- **資料夾**: `content-strategy`
- **顯示名稱**: 內容策略指南 (Content Strategy)

## 2. 描述

本 Skill 專注於品牌的「語言溝通」。它確保所有文字內容都傳達出一致的品牌個性和價值觀。

## 3. 使用時機

- **當用戶需求涉及**：
  - 撰寫網頁文案 (Copywriting)。
  - 編寫系統錯誤訊息或提示 (Microcopy)。
  - 撰寫行銷郵件或社群貼文。
  - 定義專有名詞的使用方式。

## 4. 審查清單 (Review Checklist)

在撰寫文案時，請確認：

- [ ] **語氣 (Tone)**：
  - **核心風格**: 專業 (Professional)、前衛 (Futuristic)、權威 (Authoritative).
  - **目標受眾**: 投資者、企業主、金融專業人士.
  - **關鍵字**: "即時 (Real-time)", "深入 (In-depth)", "趨勢 (Trends)", "智能 (Intelligent)".
- [ ] **內容架構**:
  - 新聞標題應簡潔有力，結論先行.
  - 數據引用需精確，避免模稜兩可的描述.
- [ ] **清晰度 (Clarity)**：
  - 是否結論先行？
  - 是否避免了過於艱澀的術語 (Jargon)？
- [ ] **一致性 (Consistency)**：
  - 專有名詞是否統一？(例如：使用 "App" 還是 "應用程式"？)
  - 標點符號使用是否規範？(全形/半形)

## 5. 格式標準

- **語言**：繁體中文 (台灣)。
- **風格**：簡潔、有力、以用戶為中心。
