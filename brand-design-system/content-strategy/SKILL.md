# 內容策略指南 (Content Strategy)

## 1. Skill 名稱

- **資料夾**: `content-strategy`
- **顯示名稱**: 內容策略指南 (Content Strategy)

## 2. 描述

本 Skill 專注於品牌的「語言溝通」。它確保所有文字內容都傳達出一致的品牌個性和價值觀。

## 3. 使用時機

- **當用戶需求涉及**：
  - 撰寫網頁文案 (Copywriting)。
  - 編寫系統錯誤訊息或提示 (Microcopy)。
  - 撰寫行銷郵件或社群貼文。
  - 定義專有名詞的使用方式。

## 4. 審查清單 (Review Checklist)

在撰寫文案時，請確認：

- [ ] **語氣 (Tone)**：
  - 是否符合品牌個性 (例如：專業、親切、創新)？
  - 是否使用了「繁體中文 (台灣)」的慣用語？
- [ ] **清晰度 (Clarity)**：
  - 是否結論先行？
  - 是否避免了過於艱澀的術語 (Jargon)？
- [ ] **一致性 (Consistency)**：
  - 專有名詞是否統一？(例如：使用 "App" 還是 "應用程式"？)
  - 標點符號使用是否規範？(全形/半形)

## 5. 格式標準

- **語言**：繁體中文 (台灣)。
- **風格**：簡潔、有力、以用戶為中心。
