# 視覺識別系統 (Visual Identity)

## 1. Skill 名稱

- **資料夾**: `visual-identity`
- **顯示名稱**: 視覺識別系統 (Visual Identity)

## 2. 描述

本 Skill 專注於品牌的「視覺呈現」。它規範了核心視覺元素的使用方式，確保所有設計產出物的一致性。

## 3. 使用時機

- **當用戶需求涉及**：
  - 設計 UI 介面 (Figma, Web)。
  - 生成圖片 (Image Generation)。
  - 選擇配色方案 (Color Palette)。
  - 使用 Logo (Logo Usage)。
  - 設定字體排印 (Typography)。

## 4. 審查清單 (Review Checklist)

在進行視覺設計時，請確認：

- [ ] **顏色 (Color)**：
  - 是否使用了品牌定義的主色 (Primary) 與輔助色 (Secondary)？
  - 顏色對比度是否符合無障礙標準 (Accessibility)？
- [ ] **字體 (Typography)**：
  - 是否使用了指定的字體家族 (Font Family)？
  - 標題 (H1-H6) 與內文 (Body) 的層級是否清晰？
- [ ] **Logo 使用**：
  - 是否使用了正確版本的 Logo (全彩/單色/反白)？
  - 是否保留了足夠的安全邊距 (Clear Space)？
  - **注意**：Logo 檔案位於 `../assets/`。

## 5. 格式標準

- **語言**：繁體中文 (台灣)。
- **輸出**：CSS 變數、Hex Codes、Tailwind Classes。
