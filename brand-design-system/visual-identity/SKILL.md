# 視覺識別系統 (Visual Identity)

## 1. Skill 名稱

- **資料夾**: `visual-identity`
- **顯示名稱**: 視覺識別系統 (Visual Identity)

## 2. 描述

本 Skill 專注於品牌的「視覺呈現」。它規範了核心視覺元素的使用方式，確保所有設計產出物的一致性。

## 3. 使用時機

- **當用戶需求涉及**：
  - 設計 UI 介面 (Figma, Web)。
  - 生成圖片 (Image Generation)。
  - 選擇配色方案 (Color Palette)。
  - 使用 Logo (Logo Usage)。
  - 設定字體排印 (Typography)。

## 4. 審查清單 (Review Checklist)

在進行視覺設計時，請確認：

- [ ] **顏色 (Color Scheme)**：
  - **背景基底**: Midnight Black (#0B0C10), Deep Navy (#1F2833).
  - **核心點綴**: Neon Blue (#45A29E), Violet (#9D50BB) - 用於按鈕發光與流線效果.
  - **文字顏色**: Web White (#E0E0E0), Muted Gray (#C5C6C7).
  - 顏色對比度是否符合無障礙標準 (Accessibility)？
- [ ] **視覺風格 (Visual Style)**：
  - **關鍵字**: Cyber-tech, Dynamic Lines, Glassmorphism, Glow Effects.
  - 是否包含「磨砂玻璃感 (Frosted Glass)」與「動態流光」？
- [ ] **字體 (Typography)**：
  - **標題 (Headings)**: Orbitron 或 Rajdhani (科技感/現代感).
  - **內文 (Body)**: Inter 或 Roboto (易讀性/專業感).
- [ ] **Logo 使用**：
  - 優先使用帶有 Neon Glow 效果的 Logo 版本.
  - 確保在深色背景上的可視性.
  - **注意**：Logo 檔案位於 `../assets/`.

## 5. 格式標準

- **語言**：繁體中文 (台灣)。
- **輸出**：CSS 變數、Hex Codes、Tailwind Classes。
